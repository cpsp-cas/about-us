# 人格与社会心理研究中心

人格与社会心理研究中心（Center for Personality and Social Psychology，CPSP）依托于中国科学院心理研究所，于2017年正式成立。中心立足中国、放眼世界，开展从基因、神经到行为、文化等各层面的整合研究，推动中国的人格与社会心理学发展，传播和普及科学心理学知识，提升人民福祉。
